Середнє з чисел

Посилання на e-olimp

Дано три різні числа a, b, c. Вивести середнє з них.

Вхідні дані
Числа a, b, c цілі та за модулем не перевищують 1000.

Вихідні дані
Вивести середнє з трьох чисел.

Пояснення

Код задачі

var  a, b, c:integer;
begin
	readln(a, b, c);
	if ((a > c)and(c > b))or((a < c)and(c < b))then writeln(c)
	else if ((a > b)and(b > c))or((a< b)and(b < c))then writeln(b)
	else writeln(a);
end.
